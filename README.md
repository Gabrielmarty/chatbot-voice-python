# 🎤 Voice Chatbot com IA

Assistente por voz usando Python que escuta, entende e responde com voz.

## 🚀 Tecnologias

* Python
* OpenAI API (Whisper + ChatGPT)
* gTTS
* Pygame

## 🔄 Funcionamento

1. Usuário fala
2. Áudio vira texto (Whisper)
3. ChatGPT responde
4. Resposta vira voz

## ▶️ Como rodar

pip install -r requirements.txt
python main.py

## 🔐 Configuração

Crie um arquivo `.env`:

OPENAI_API_KEY=sua_chave_aqui

## 🛑 Para sair

Diga: **sair**
